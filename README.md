# VUEJS CALENDAR
## Опис
Простий календар написаний на vue.js у стилі Options API. Дозволяє створювати та видаляти події, прив'язані до конкретних дат.
## Технології
- JS
- CSS
- Vue.js
- Apache (XAMMP)
## Функціонал
- Додавання подій до дати
- Видалення подій
## Запуск проєкту
1. Встановить XAMMP.
2. Скопіюйте проєкт у папку htdocs.
3. Запустіть Apache через панель керування XAMMP.
4. Відкрийте браузер та перейдіть за адресою: https://localhost/vuejs_calendar
